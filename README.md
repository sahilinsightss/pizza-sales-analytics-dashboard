# pizza-sales-analytics-dashboard
Interactive Power BI dashboard analyzing pizza sales, revenue trends, and operational insights.

## Project Overview
This project analyzes pizza sales data using Power BI to identify revenue trends, customer ordering behavior, and operational insights.

## Tools Used
- SQL
- Power BI
- Excel

## Dashboard Features
- KPI cards for revenue, orders, and average order value
- Monthly revenue trend analysis
- Peak order hour analysis
- Category-wise revenue contribution
- Top-selling pizzas visualization
- Interactive slicers

## Key Insights
- Peak ordering activity occurred during afternoon and evening hours.
- Certain pizza categories consistently generated higher revenue.
- Top-selling pizzas contributed significantly to overall sales performance.
- Monthly trends revealed fluctuations in customer demand.
